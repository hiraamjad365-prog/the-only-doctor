# The Only Doctor: Blackwood's Physician

A text-based, branching narrative game built in Python for the Stanford Code in Place 2026 Final Project Showcase.

## The Story
You are the sole physician in Blackwood, a small mountain town completely cut off from civilization by a brutal winter storm. With zero backup and depleted supplies, you are the final authority on every medical and moral crisis that walks through your clinic door.

## Features
* Branching Decisions: Choose how to handle tough moral dilemmas (like rationing scarce vaccines).
* Dynamic State Tracking: Every decision shifts a hidden Integrity Score that tracks your ethical standing.
* Modular Code Architecture: Clean separation of game state and core logic execution.
